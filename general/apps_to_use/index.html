Recommended Apps and Software for macOS

Media Players

IINA: The Modern Media Player for macOS

  • Free and open-source: IINA

  • An excellent QuickTime alternative.

  • Install using Homebrew:

bash brew install --cask iina

Screen Recording

IShowU Instant

  • Real-time screen recording for macOS.
  • Official site: IShowU Instant

Video Downloaders

Downie

  • Download videos from the web with ease.
  • Includes browser plugins for convenience.
  • Official site: Downie

Media Converters

Permute

  • Convert media files to various formats.
  • Official site: Permute

Virtual Machines (VMs)

  • *UTM*: A macOS-specific VM solution. UTM
  • *VirtualBox*: A cross-platform VM solution.
  • *KVM*: Preferred for Linux environments.

Productivity and Research Tools

JupyterLab

  • Enhance JupyterLab with extensions like `nbslide` for creating slideshows.

  • Install `nbslide` via pip:

bash pip install nbslide

  • More details: nbslide on PyPI

System Monitoring Tools

  • *htop*: Interactive process viewer.
  • *btop*: Advanced visual information compared to `top` and `htop`. Highly recommended.

Document Management

OpenPaper

  • An alternative to DEVONthink for scanning and organizing personal documents.
  • Official site: OpenPaper

EasyFind

  • Freeware from DEVONtechnologies.
  • Advanced search capabilities without requiring indexing.
  • Official site: EasyFind

Paperless-ngx

  • Document management system to create a searchable online archive of physical documents.
  • Alternative to DEVONthink.
  • GitHub repository: Paperless-ngx
