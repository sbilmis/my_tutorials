Report Template

This section provides a basic LaTeX template for creating a report with customized margins, line spacing, and optional page numbering control. This template is useful for general reports and internal documents.

\documentclass{article}                   % Use the 'article' document class for standard reports
\usepackage[utf8]{inputenc}               % Enable UTF-8 encoding, allowing special characters

\usepackage[margin=1.4in]{geometry}       % Set 1.4-inch page margins on all sides for a balanced layout
\usepackage{setspace}                     % Load the 'setspace' package to control line spacing
\renewcommand{\baselinestretch}{1.5}      % Increase line spacing to 1.5, which improves readability

\pagenumbering{gobble}                    % Suppress page numbering by setting it to 'gobble'

\begin{document}                          % Begin the content of the document

% Document content goes here

\end{document}                            % End the document environment

Table Template

This table template provides a clean and professional layout for displaying data. It uses the `booktabs` package for high-quality table lines and includes settings for row and column spacing.

\usepackage{booktabs}                     % Use 'booktabs' package for professional-quality table lines

\begin{table*}[t]                         % Start a table that spans the full page width and is positioned at the top
  \centering                              % Center the table horizontally
  \renewcommand{\arraystretch}{1.4}       % Increase row height for readability; 1.4 is typically comfortable
  \setlength{\tabcolsep}{7pt}             % Set horizontal padding between columns to 7pt for clear separation

  \begin{tabular}{cc}                     % Define a table with two centered columns
    \toprule                              % Add a clean, thick top line to the table (requires 'booktabs' package)

    $T_{bb}$ States             & Molecular Picture Contents \\ % Column headers describing table content
    \midrule                              % Add a middle rule for separation between header and data rows

    $T_{bb}^{- -}$  & $B^- B^{*-}$ \\     % First row: Display $T_{bb}^{--}$ state and its molecular components
    $T_{bb}^{-}$    & $B^0 B^{*-}$, $B^{-} B^{*0}$ \\ % Second row with multiple components
    $T_{bb}^{0}$    & $B^0 B^{*0}$ \\     % Third row: $T_{bb}^{0}$ state and its molecular composition
    $T_{bb(s)}^{-}$ & $B_s^0 B^{*-}$, $B_s^{*0} B^{-}$ \\ % Fourth row with $T_{bb(s)}^{-}$ state components
    $T_{bb(s)}^{0}$ & $B_s^0 B^{*0}$, $B^{*0} B_s^{*0}$ \\ % Fifth row with $T_{bb(s)}^{0}$ state
    $T_{bb(ss)}^{0}$ & $B_s^0 B_s^*$ \\   % Sixth row: $T_{bb(ss)}^{0}$ state and its components
    \bottomrule                           % Add a clean, thick bottom line to the table

  \end{tabular}

  \caption{Possible states of $T_{bb}$ with $J^{P} = 1 ^{+}$.} % Provide a caption for context in the document
  \label{tab:1}                         % Label the table for referencing it within the document
\end{table*}
