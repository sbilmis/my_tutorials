Git Overview

  • Git is a distributed version control system that tracks changes in files and coordinates work among multiple people. It's widely used for source code management in software development and for tracking changes in any set of files.
  • Commonly used platforms for Git repositories include GitHub, GitLab, and Bitbucket, which also offer additional tools for collaboration and project management.

.gitignore

  • `.gitignore` files tell Git to ignore certain files and directories in the working directory, so they don't get tracked in the repository. Place the `.gitignore` file in the root of your working directory (not inside the `.git` directory).

  • See the GitHub gitignore repository for pre-made `.gitignore` templates for various languages and applications, including LaTeX.

  • Example `.gitignore` file for LaTeX projects:

``` text ## Place this file in your working directory ## Common auxiliary files generated by LaTeX and pdflatex .aux .lof .log .lot .fls .out .toc .fmt .fot .cb .cb2 ..lb

## Intermediate documents .dvi .xdv -converted-to.

## Generated if empty string is given at "Please type another file name for output:" .pdf

## Bibliography auxiliary files .bbl .bcf .blg -blx.aux -blx.bib .run.xml

## Build tool auxiliary files .fdb_latexmk .synctex .synctex(busy) .synctex.gz .synctex.gz(busy) .pdfsync

## Build tool directories for auxiliary files latex.out/

## Files from other LaTeX packages, editors, and systems .alg .loa .thm .nav .pre .snm .vrb .soc .cut .cpt .spl .ent .lox .mf .mp .t[1-9] .tfm .acn .acr .glg .glo .gls *.glsdefs # Additional package-specific extensions...

## Editors: # LyX .lyx~ # Kile .backup # WinEdt .bak .sav # Gummi ..swp # KBibTeX ~[0-9] # Texpad .texpadtmp # Emacs AUCTeX auto folder ./auto/ .el ```
