📄 lesson.org

#+EXPORT_FILE_NAME: episode1.md

title: "Introduction to Carpentries-style Lessons" teaching: 10 exercises: 5 questions: - "What is the purpose of this tutorial?" - "How can I publish it on GitHub Pages?" objectives: - "Set up a Carpentries-style lesson" - "Write and export Org-mode content to Markdown" keypoints: - "You can write tutorials in Org-mode and export them to Markdown" - "Markdown files go into the 'episodes/' folder in your Carpentries lesson"

Introduction

Solution

To run this Python code, use a terminal or Jupyter Notebook.

python script.py

Next Steps

  • Write more `.org` files like this.
  • Export them to `.md` and place them in your `episodes/` directory.

⸻

🛠️ How to Use This

  1. Save this as lesson.org
  2. Open in Emacs
  3. Export to Markdown:

C-c C-e m m

→ This creates episode1.md

  1. Place episode1.md in your Tutorials/episodes/ folder
  2. Push to GitHub --- GitHub Pages will rebuild automatically (if your template is working)

✅ After This
