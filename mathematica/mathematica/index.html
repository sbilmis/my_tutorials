Patterns (TBO)

  • `Blank[]` - Single unspecified object `x_`
  • `BlankSequence[]` - Sequence of objects `x__`
  • `BlankNullSequence[]` - Null sequence of objects `x___`
  • `Pattern[sym, obj]` or `sym:obj` - Creates a pattern where `sym` must match `obj`

Input Expressions

  • These expressions are equivalent in Mathematica:

wolfram Mean[{1, 2, 3}] Mean@{1, 2, 3} {1, 2, 3} // Mean

Accessing Parts of an Expression

  • The following expressions are equivalent:

wolfram Part[{x, y, z}, 2] {x, y, z}[[2]]

String Operations

  • Concatenate strings:

wolfram StringJoin["Hello", "World"] (* Output: HelloWorld *)

Removing Template Notebook Header

  • To remove the docked header in the notebook:

wolfram SetOptions[EvaluationNotebook[], DockedCells -> {}]

Working with Lists

  • Remove duplicates from a list:

wolfram DeleteDuplicates[{1, 7, 8, 4, 3, 1, 9, 9, 8, 2}]

  • Count distinct elements in a list:

wolfram CountDistinct[{1, 7, 8, 4, 3, 1, 9, 9, 8, 2}]

Determining Variable Types

  • Use `Head` to check the type of a variable:

``` wolfram list = {1, 2, 3}; integer = 1; real = 0.1;

Head[list] Head@integer Head@real ```

Exporting Data to a File

  • Example to export data to a file:

wolfram s01 = 10; myDataS01 = Table[{i, i^2}, {i, 1, 10}]; Export["./results/ground_mass_etac_s0_" <> ToString[s01] <> "fit_ground.dat", myDataS01]

  • Create a directory and export data if it doesn't exist:

wolfram SetDirectory[NotebookDirectory[]] path = "./test/"; myDataS01 = Table[{i, i^2}, {i, 1, 10}]; Export[path <> "test.dat", myDataS01]

Importing Data and Plotting

  • Install MaTeX for LaTeX typesetting in Mathematica plots.

wolfram << MaTeX` ConfigureMaTeX[ "pdfLaTeX" -> "/Library/TeX/texbin/pdflatex", "Ghostscript" -> "/usr/local/bin/gs" ]

  • Import data:

wolfram SetDirectory[NotebookDirectory[]] data1 = Import["./results/ground_mass_etac_s0_9fit_ground.dat"]; data2 = Import["./results/ground_mass_etac_s0_10fit_ground.dat"]; data3 = Import["./results/ground_mass_etac_s0_11fit_ground.dat"];

  • Apply a cut to the data:

wolfram pdata1 = Select[data1, 7.0 <= #[[1]] <= 14.0 &];

Custom Plotting Options

  • Set plot options and LaTeX labels:

``` wolfram texStyle = {FontFamily -> "Latin Modern Roman", FontSize -> 12};

myPlotOptions1 = { PlotTheme -> {"Scientific"}, PlotLegends -> { MaTeX["s_0 = 9~\rm{GeV}^2", Magnification -> 1.2], MaTeX["s_0 = 10~\rm{GeV}^2", Magnification -> 1.2], MaTeX["s_0 = 11~\rm{GeV}^2", Magnification -> 1.2] }, FrameLabel -> MaTeX[{"M^2~\rm{GeV^2}", "m_{\eta_c}"}, Magnification -> 1.2], PlotRange -> {{7, 14}, {1.0, 3.0}} }

ListLinePlot[{pdata1, pdata2, pdata3}, Evaluate[myPlotOptions1], BaseStyle -> texStyle] ```

Fitting, Plotting Residuals, and Calculating Chi-Square

  • Fit a model to data, plot residuals, and calculate $\chi^2$:

``` wolfram data = {{0, 2.52}, {1., 2.83}, {2., 3}, {3., 3.2}, {4.1, 3.35}, {5., 3.47}, {6, 3.57}, {7, 3.66}}; model = k x + b Tanh[c (x - x0)] + d; fit = NonlinearModelFit[data, model, {{k, .1}, b, c, d, {x0, 20}}, x];

Plot[fit[x], {x, 0, 25}, Epilog -> {{Red, PointSize[.01], Point[data]}, Inset[fit["ParameterTable"], {9, 9}]}, Frame -> True, PlotLabel -> fit[x]] ```

Solving Equations

  • Use `Solve` and `Reduce` for equation solving, with assumptions and domains:

wolfram Solve[x^2 + y^2 == 4 && 2 x - 2 == 0, {x, y}, Assumptions -> y > 0 && x < 0] Solve[x^2 == -1, x, Complexes]

Data Concatenation

  • Concatenate two lists `{x1, x2, x3}` and `{y1, y2, y3}` as `{{x1, y1}, {x2, y2}}`:

wolfram Thread[{a, b, c}, {x, y, z}] (* Output: {f[a, x], f[b, y], f[c, z]} *)

Setting Default External Evaluator for Python

  • Find available Python versions:

wolfram FindExternalEvaluators["Python"]

  • Set a specific Python version:

wolfram RegisterExternalEvaluator["Python", "/usr/local/bin/python3.9"]
