What is Zotero?

  • Zotero is a free, open-source tool for managing references, citations, and research materials. It allows users to collect, organize, cite, and share research using its desktop application, browser connector, and cloud sync.

WebDAV Sync

  • Zotero supports syncing libraries via WebDAV, which allows users to store their attachments on a remote server instead of Zotero's cloud storage.
  • pcloud and koofr have webdav service.
  • Be cautious when using WebDAV with multiple profiles, as it may cause sync issues if different folders are used.

Using Multiple Profiles

  • Zotero allows the creation of multiple profiles for managing different libraries or accounts. Use the `-P` option when launching Zotero to choose a profile:

shell zotero -P

  • Example: The `sbilmis~mybooks~` profile was created using the account `sbilmis@outlook.com`.

  • *Caution*: Using multiple profiles with WebDAV sync may lead to issues if WebDAV storage does not support separate folders for different profiles.

Addons for Zotero

Official Zotero Plugin Directory provides a comprehensive list of supported and tested plugins. Here I list that I commonly use.

  • Zotero Addons Manager:
  • Install and manage Zotero addons easily, including version-specific addons for different Zotero versions.
  • Zotero Better Notes:
  • Organize and annotate notes directly within Zotero to improve research note management.
  • ZotFile:
  • Enhance file management by renaming, moving, and syncing PDFs with tablets.
  • Zotero Folder Import:
  • Import a whole folder of files into Zotero.
  • Zotero Storage Scanner:
  • Scan Zotero storage for missing or unlinked files.
  • Zotxt:
  • Integrate Zotero with Emacs for inserting citations into documents.
  • Zutilo:
  • Add functionalities like copying file paths, batch editing, and shortcut enhancements.
  • Zotero Inspire:
  • Integrate Zotero with INSPIRE-HEP for physics-related literature.
  • Zotero Preview:
  • Add a preview pane to view bibliography entries directly within Zotero.
  • Zotero OCR:
  • Add OCR capabilities to Zotero for extracting text from scanned PDFs.

Zotxt: Emacs Integration for Zotero

zotxt{.verbatim} is an essential add-on that connects Zotero with Emacs, enabling seamless citation management. It allows you to search for references and insert them directly into your documents while working in Emacs.

*Installation Steps*

  1. *Install zotxt Add-on in Zotero*:
    • Download the plugin from the official repository: zotxt GitHub Repository.
    • In Zotero, navigate to Tools > Add-ons > Install Add-on From File{.verbatim} and select the downloaded .xpi file.
    • Restart Zotero to activate the plugin.

  2. *Set Up zotxt in Emacs*:

    • Add the following configuration to your Emacs init file (init.el):

    {.commonlisp org-language="emacs-lisp"} (use-package zotxt :ensure t :config (setq zotxt-default-bibliography-style "apa"))

    • Reload your configuration or restart Emacs.
    • *Usage*:
    • Search for references using: M-x zotxt-select.
    • Insert a citation into your text with: M-x zotxt-insert-reference.

Zotero Actions & Tags (AT): Automate and Organize Your Workflow

Actions & Tags (AT), also known as Zotero Tag, is a powerful plugin for Zotero that helps you automate tasks and organize your workflow using tags and scripts.

*Key Features* AT provides the following capabilities:

  1. *Automatic Tagging*:
    • Automatically tag Zotero items based on events like new items added or user-defined shortcuts.
  2. *Custom Scripts for Automation*:
    • Replace tags dynamically.
    • Copy item links quickly.
    • Auto-generate notes when opening an item.
    • Auto-translate the title or abstract of newly added items.

*Installation Steps*

  1. Download the plugin from the official GitHub repository: Zotero Actions Tags GitHub Page.
  2. Install the add-on in Zotero:
    • Go to Tools > Add-ons > Install Add-on From File{.verbatim}.
    • Select the downloaded .xpi file and restart Zotero.

*Example Use Cases*

  • *Replace Tags*: Update an outdated tag with a new one in bulk.
  • *Copy Item Link*: Copy a Zotero item\'s link for quick sharing or referencing.
  • *Auto-Generate Notes*: Automatically generate a note when you open a Zotero item to add annotations.
  • *Auto-Translate*: Automatically translate the title or abstract of newly added items using predefined scripts.

*Example Workflow*:

  1. Add a tag auto-note to an item.
  2. Open the item → a note is automatically generated.
  3. Use shortcut Ctrl+Shift+C to copy the item link.

*Benefits*:

  • Save time by automating repetitive actions.
  • Maintain consistency in tagging and metadata management.
  • Integrate seamlessly with other tools like Emacs or Markdown editors.
